## Pré-requisitos

- [Node.js](https://nodejs.org/) (v18+)
- [npm](https://www.npmjs.com/) (v9+)

## Instalação

Clone o repositório:
git clone {link-do-repositorio}

Navegue até o diretório do projeto:
cd {nome-do-diretorio}

Instale as dependências (se necessário):
npm install

## Execução

node index.js
